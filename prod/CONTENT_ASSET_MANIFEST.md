### Image Optimization

**Tools:**
- TinyPNG/TinyJPG (https://tinypng.com/) - Compression
- Squoosh (https://squoosh.app/) - Format conversion
- ImageMagick - Batch processing

**Process:**
1. Collect high-resolution original photos
2. Crop/resize to specified dimensions
3. Optimize file size (use 80-85% JPEG quality)
4. Convert to WebP for modern browsers (optional)
5. Verify file sizes meet limits
6. Test on actual pages

### Document Creation

**Tools:**
- Adobe InDesign/Illustrator - Professional brochures
- Canva - Quick branded documents
- Microsoft Word/PowerPoint - Basic documents

**Process:**
