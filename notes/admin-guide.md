## Content Management

### News & Articles

#### Creating a News Article

1. Go to **Content → News**
2. Click **"New Article"**
3. Fill in required fields:
   - **Title** - Article headline (50-100 chars recommended)
   - **Slug** - URL-friendly version (auto-generated from title)
   - **Category** - Select: Projects, Equipment, Awards, Company News
   - **Excerpt** - Short summary for listings (150-200 chars)
   - **Content** - Full article content (use rich text editor)
   - **Hero Image** - Featured image (1200x630px recommended)
   - **Author** - Your name
   - **Tags** - Keywords for searchability
4. **Status:**
   - Draft - Not visible to public
   - Published - Live on website
   - Archived - Hidden but saved
5. Click **"Save"** or **"Publish"**

#### Editing Articles

1. Go to **Content → News**
2. Click article title or **"Edit"** button
3. Make changes
4. Click **"Update"**

#### Deleting Articles

1. Go to **Content → News**
2. Click **"Delete"** on article
3. Confirm deletion

**Note:** Deleted articles cannot be recovered.

---

### Job Postings

#### Creating Job Posting

1. Go to **Content → Jobs**
2. Click **"New Job"**
3. Required fields:
   - **Job Title** - Position name
   - **Department** - Operations, Engineering, Admin, Safety, HR
   - **Location** - Office location
   - **Employment Type** - Full-time, Part-time, Contract
   - **Experience Level** - Entry, Mid, Senior, Executive
   - **Description** - Full job description
   - **Responsibilities** - Bullet points
   - **Qualifications** - Required qualifications
   - **Skills** - Technical and soft skills
   - **Salary Range** - Optional
   - **Application Deadline** - Date
4. Click **"Publish Job"**

#### Managing Applications

1. Go to **Submissions → Job Applications**
2. View all applications by job
3. Actions per application:
   - **View Details** - See full application
   - **Download Resume** - Get PDF/DOC
   - **Update Status** - Mark as: New, Reviewing, Shortlisted, Rejected, Hired
   - **Add Notes** - Internal comments
4. Contact candidates via email listed

---

### Projects Portfolio

#### Adding Project

1. Go to **Content → Projects**
2. Click **"New Project"**
3. Fill in:
   - **Project Name**
   - **Client** - Company name
   - **Service Type** - Primary service provided
   - **Location** - Project site
   - **Duration** - Start and end dates
   - **Description** - Project overview
   - **Challenges** - Problems solved
   - **Results** - Outcomes and metrics
   - **Images** - Project photos (min 3, up to 10)
   - **Status** - Completed, Ongoing, Upcoming
4. Click **"Save Project"**

## Managing Submissions

### Contact Form Submissions

1. Go to **Submissions → Contact**
2. View all inquiries
3. Columns:
   - Name, Email, Phone
   - Category, Subject
   - Date submitted
   - Status
4. Actions:
   - **View** - See full message
   - **Reply** - Send email (opens mail client)
   - **Mark as Resolved**
   - **Delete**

**Best Practice:** Respond within 24 hours

---

### Quote Requests

1. Go to **Submissions → Quotes**
2. View quote details:
   - Contact information
   - Service requested
   - Project details
   - Timeline and budget
3. Actions:
   - **Assign to Sales** - Forward to sales team
   - **Create Quote** - Generate formal quote
   - **Mark as Won/Lost**

---

### Partnership Inquiries

1. Go to **Submissions → Partnerships**
2. Review proposals
3. Actions:
   - Schedule meeting
   - Request more info
   - Accept/Decline proposal

---

### Supplier Applications

1. Go to **Submissions → Suppliers**
2. Review company details and capabilities
3. Actions:
   - Approve - Add to supplier database
   - Request documentation
   - Reject with reason

## Media Library

### Uploading Images

1. Go to **Media → Images**
2. Click **"Upload Images"**
3. Select files (JPG, PNG, WebP)
4. Add:
   - Title - Descriptive name
   - Alt Text - For accessibility
   - Caption - Optional description
   - Category - For organization
5. Click **"Upload"**

**Image Guidelines:**
- Max size: 5MB per image
- Recommended: 1920x1080px or higher
- Optimize before upload (use TinyPNG)
- Use descriptive filenames

---

### Managing Documents

1. Go to **Media → Documents**
2. Upload PDFs:
   - Brochures
   - Reports
   - Certificates
   - Case Studies
3. Organize into categories:
   - Services
   - Company
   - Press Releases
   - Newsletters

**Document Guidelines:**
- Max size: 10MB
- Professional formatting
- Include YPI branding
- Use clear filenames

## Best Practices

### Content Publishing

1. **News Articles:**
   - Publish at least twice monthly
   - Include high-quality images
   - Use proper categories and tags
   - Proofread before publishing
   - Add internal links to services/projects

2. **Job Postings:**
   - Update regularly (weekly review)
   - Remove filled positions promptly
   - Use clear, specific job titles
   - Include salary ranges when possible
   - Set realistic application deadlines

3. **Projects:**
   - Showcase best work first
   - Include client testimonials (with permission)
   - Use professional photos
   - Quantify results (metrics, percentages)
   - Update portfolio quarterly

### Form Management

1. **Response Times:**
   - Contact forms: 24 hours
   - Quote requests: 48 hours
   - Job applications: 5 business days
   - Partnership inquiries: 3 business days

2. **Organization:**
   - Check submissions daily
   - Archive resolved items weekly
   - Follow up on pending items
   - Keep notes on each submission

### Media Management

1. **Image Optimization:**
   - Compress before upload
   - Use appropriate formats (JPG for photos, PNG for graphics)
   - Add alt text for all images
   - Organize in categories

2. **Document Standards:**
   - Use consistent branding
   - Professional design
   - Update documents annually
   - Version control (add dates to filenames)

## Troubleshooting

### Cannot Login

**Problem:** Forgot password or login fails

**Solution:**
1. Contact system administrator to reset password
2. Check email for any typos
3. Ensure cookies are enabled in browser
4. Try incognito/private mode

---

### Content Not Saving

**Problem:** Changes not being saved

**Solution:**
1. Check internet connection
2. Refresh page and try again
3. Check browser console for errors (F12)
4. Try different browser
5. Contact technical support if persists

---

### Images Not Uploading

**Problem:** Image upload fails

**Solution:**
1. Check file size (must be under 5MB)
2. Verify file format (JPG, PNG, WebP only)
3. Try compressing image
4. Check internet connection
5. Try uploading one at a time

---

### Email Notifications Not Received

**Problem:** Not receiving form submission emails

**Solution:**
1. Check spam/junk folder
2. Verify email settings in **Settings**
3. Test with a sample submission
4. Check with technical team if email service is configured
5. Add noreply@yellowpowerinternational.com to safe senders

---

### AI Features Not Working

**Problem:** PowerBot or AI Search not responding

**Solution:**
1. Check if AI features are enabled in Settings
2. Verify with technical team that API keys are configured
3. Check browser console for errors
4. Try refreshing the page
5. Contact support if problem persists
